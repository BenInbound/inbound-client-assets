Technical Notes

Validation Tools

  • Google Rich Results Test: https://search.google.com/test/rich-results
  • Schema.org Validator: https://validator.schema.org/
  • Structured Data Testing Tool: Google Search Console

Best Practices

  • Use JSON-LD format (preferred by Google)
  • Keep schema data accurate and up-to-date
  • Avoid markup of non-visible content
  • Follow schema.org specifications strictly
  • Test before deploying to production

Common Issues

  • Missing required properties
  • Incorrect data format (dates, prices)
  • Duplicate or conflicting markup
  • Misleading content in schema
