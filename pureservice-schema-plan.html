Critical Schema Types

The following schema types are critical for Pure Service's business visibility and SEO performance:

Organization Schema (Priority 1)

Missing Critical

Establishes the company's basic identity and contact information for search engines.

  • Company name and legal structure
  • Contact information (phone, email, address)
  • Social media profiles
  • Logo and brand imagery
  • Company URLs and alternate domains

LocalBusiness Schema (Priority 1)

Missing Critical

Enables the business to appear in local search results and Google Maps.

  • Business location(s)
  • Hours of operation
  • Service areas and radius
  • Contact methods
  • Pricing information when applicable

Service Schema (Priority 1)

Missing Critical

Describes individual services offered by Pure Service for better search visibility.

  • Service name and description
  • Service category classification
  • Area served by the service
  • Price and availability
  • Service provider information

Review/AggregateRating Schema (Priority 2)

Missing Important

  • Average rating and count

Breadcrumb Schema (Priority 2)

WebPage/Article Schema (Priority 2)

Technical Implementation Details

JSON-LD Format Preference

All Schema.org markup will be implemented using JSON-LD format, placed in the <head> section of HTML pages. This approach is:

  • Search engine recommended and preferred
  • Easier to maintain and update
  • Independent of page HTML structure
  • Supports complex nested data

Organization Schema Example

{ "@context": "https://schema.org/", "@type": "Organization", "name": "Pure Service", "url": "https://www.pureservice.com", "logo": "https://www.pureservice.com/logo.png", "description": "Professional service provider", "telephone": "+1-XXXX-XXXX", "email": "contact@pureservice.com", "address": { "@type": "PostalAddress", "streetAddress": "Address", "addressLocality": "City", "addressRegion": "State", "postalCode": "ZIP", "addressCountry": "US" }, "sameAs": [ "https://www.facebook.com/...", "https://www.linkedin.com/...", "https://www.twitter.com/..." ] }

LocalBusiness Schema Example

{ "@context": "https://schema.org/", "@type": "LocalBusiness", "name": "Pure Service", "image": "https://www.pureservice.com/image.jpg", "description": "Professional service provider", "address": { "@type": "PostalAddress", "streetAddress": "Address", "addressLocality": "City", "addressRegion": "State", "postalCode": "ZIP", "addressCountry": "US" }, "telephone": "+1-XXXX-XXXX", "openingHoursSpecification": { "@type": "OpeningHoursSpecification", "dayOfWeek": ["Monday", "Tuesday", "Wednesday", "Thursday", "Friday"], "opens": "09:00", "closes": "17:00" }, "priceRange": "$", "areaServed": ["City1", "City2", "State"] }

Service Schema Example

{ "@context": "https://schema.org/", "@type": "Service", "name": "Service Name", "description": "Detailed service description", "provider": { "@type": "LocalBusiness", "name": "Pure Service" }, "areaServed": ["City1", "City2", "State"], "availableChannel": { "@type": "ServiceChannel", "serviceUrl": "https://www.pureservice.com/service" } }
